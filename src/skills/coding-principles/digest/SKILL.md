# Coding Principles Digest

## Purpose
Use this skill as the default working baseline for the ten TripleS coding principles. It replaces eagerly reading all ten full principle files.

## Procedure
1. Read `references/digest.md` once — it names every principle and its one-sentence rule.
2. Apply them by default while implementing.
3. Open the full `coding-principles/<name>` skill only when a judgment call is genuinely contested or a reviewer disputes it.
