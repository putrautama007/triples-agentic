# Testing Digest

## Purpose
Use this skill as the default testing baseline for implementation and QA-check agents. It condenses `testing-strategy` + `testing-types` into a single reference.

## Procedure
1. Read `references/digest.md` for the pyramid, test-type definitions, tools, and anti-patterns.
2. Apply it while writing or reviewing tests.
3. Open the full `quality/testing-strategy` or `quality/testing-types` skill only when you need the complete depth (e.g. designing a full suite).
